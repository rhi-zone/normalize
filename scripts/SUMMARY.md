Shell scripts for development and maintenance tasks. `pre-commit` is the git pre-commit hook — it runs `cargo fmt --check`, `cargo clippy`, and `normalize rules run` (all engines: syntax, native, fact; `severity=error` blocks commits). `missing-grammars.sh` queries the crates.io API to find arborium language features not yet implemented in `normalize-languages`. `session-corrections.sh` extracts correction patterns from Claude Code session logs to surface candidates for new `CLAUDE.md` rules. `validate-calls-scm.sh` validates capture names in all `.calls.scm` query files against the allowed set (`@call`, `@call.write`, `@call.qualifier`), outputting SARIF 2.1.0; exits 1 on violations for CI use.
