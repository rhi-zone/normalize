# scripts/

Shell scripts for development and maintenance tasks. `pre-commit` is the git pre-commit hook — it runs `cargo fmt --check`, `cargo clippy`, and normalize's own rule checks (`severity=error` blocks commits); it runs the `syntax` and `native` engines only (the `fact` engine is skipped due to a known malloc crash in the ascent-interpreter/abi_stable boundary, tracked as a 0.3.0 bug). `missing-grammars.sh` queries the crates.io API to find arborium language features not yet implemented in `normalize-languages`. `session-corrections.sh` extracts correction patterns from Claude Code session logs to surface candidates for new `CLAUDE.md` rules.
