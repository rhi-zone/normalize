# normalize-ecosystems/src

Source for the normalize-ecosystems crate.

`lib.rs` defines the `Ecosystem` trait, all shared data types, and the default `query()` / `detect_tool()` / `find_tool()` implementations. `ecosystems/` contains one module per ecosystem. `cache.rs` provides the on-disk JSON cache keyed by ecosystem + package name (and `cache_base()` → `~/.cache/normalize/`). `http.rs` provides a thin ureq wrapper with gzip support used by the registry-fetching ecosystem impls; `get_bytes` downloads a URL to raw bytes for binary artifacts.

`source_archive.rs` downloads and extracts a package's source archive (`.zip` via `zip`, `.tar.gz` via `flate2`+`tar`) into `~/.cache/normalize/sources/{ecosystem}/{pkg}/{version}/`, staging via a tempdir and atomic rename (cache-hit skip if the dir exists). `doc_tree.rs` extracts a `SymbolDoc` from an extracted source tree: it parses supported files with `parse_with_grammar`, walks to the definition node whose name matches the last segment of `symbol_path`, and pulls docstring/signature via the `normalize-languages` `Language` trait (`doc_format: PlainText`). Both are shared infrastructure consumed by per-ecosystem (Go, Python) doc extractors.

`symbol_docs.rs` defines `SymbolDoc` (structured docs for the `normalize docs` command) and `DocFormat` (`Markdown`/`Rst`/`Html`/`PlainText`). Doc bodies are stored **source-native**: the body lives in `doc_body` tagged with `doc_format`, and rendering to display Markdown happens at the output layer (`render_symbol_doc` in the main crate), never at fetch time. `local_docs.rs` extracts Rust `///` docs (→ `DocFormat::Markdown`); `docs_rs.rs` fetches rustdoc docblocks as raw HTML fragments (→ `DocFormat::Html`) and exposes the `html_to_markdown` / `strip_html_tags` / `decode_html_entities` / `normalize_blank_lines` helpers publicly so the output layer can render them. `SymbolDoc::kg_id()` derives its prefix from `language` (`docs-rust-…`).

`go_docs.rs` (gated `#[cfg(feature = "go")]`) provides Go symbol docs. `GoLocalDocsExtractor` resolves an import path to its on-disk source via `normalize-local-deps`' `GoDeps::resolve_external_import` (module cache `$GOMODCACHE`, or the SDK `$GOROOT/src` tree for stdlib), reads the version from the `repo@v1.2.3` cache-dir name, and delegates to `doc_tree::extract_from_source_tree`. `GoRemoteDocsFetcher` resolves the version (explicit, or `@latest` from `proxy.golang.org`), builds the proxy `{module}/@v/{version}.zip` URL — applying Go's `!`-lowercase escaping for uppercase letters via `escape_go_proxy` — downloads/extracts via `source_archive::fetch_and_extract` (the zip's `{module}@{version}/` top dir is handled transparently by the recursive walk), and extracts the symbol. Stdlib modules have no proxy source, so the fetcher returns a clear `NotFound`. The `Go` ecosystem's `package_from_symbol` splits on `#` (`import/path#Symbol`) or, failing that, the last `.` (`fmt.Println` → `("fmt","Println")`); a bare symbol with neither returns `None`.

`python_docs.rs` (gated `#[cfg(feature = "python")]`) provides Python symbol docs. `PythonLocalDocsExtractor` resolves a top-level import name to its on-disk source via `normalize-local-deps`' `PythonDeps::resolve_external_import` (stdlib `lib/pythonX.Y`, or the project venv's `site-packages`); a package resolves to a directory (`requests/`) or a single-file module (`six.py`, whose parent dir is searched), and the installed version is recovered from a matching `{name}-{version}.dist-info` directory (PEP 503-normalized name match) or left empty for stdlib. `PythonRemoteDocsFetcher` resolves the version (explicit, or latest from the PyPI JSON `info.version`), finds the sdist via `pypi_sdist_url` (scans `urls[]` for `packagetype == "sdist"`, picking `ArchiveKind` from the filename extension — `.zip` → `Zip`, else `TarGz`), downloads/extracts via `source_archive::fetch_and_extract`, and extracts the symbol from the `{pkg}-{version}/` tree (recursive walk handles `src/` layouts). The `Python` ecosystem's `package_from_symbol` takes the first `.`-segment as the package and the dotted remainder as the symbol path (`requests.adapters.HTTPAdapter` → `("requests", "adapters.HTTPAdapter")`); a bare name with no `.` returns `None`.
