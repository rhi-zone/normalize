# normalize-typegen/tests/snapshots

Insta snapshot files for typegen backend output.

20 `.snap` files covering generated code for all backends: TypeScript types (user, readonly), Zod (user, with-types), Valibot (user, with-types), Python dataclasses (user, frozen), TypedDict (user), Pydantic (user, frozen), Go structs (user, no-tags), Rust structs (user, no-serde), and tagged union outputs for Rust/TypeScript/Zod. Also includes OpenAPI Petstore output for TypeScript and Zod backends.
