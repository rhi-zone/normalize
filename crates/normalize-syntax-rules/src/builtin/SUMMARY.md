# normalize-syntax-rules/src/builtin

Embedded builtin rule files, compiled into the binary via `include_str!`.

`mod.rs` declares the `BUILTIN_RULES` constant (a `&[BuiltinRule]` slice) that references all 45 `.scm` rule files. Rules are organized by language namespace: Rust (11 rules: `todo-macro`, `println-debug`, `dbg-macro`, `expect-empty`, `unwrap-in-impl`, `unnecessary-let`, `unnecessary-type-alias`, `chained-if-let`, `numeric-type-annotation`, `tuple-return`, `static-mut`), JavaScript (7: `console-log`, `unnecessary-const`, `module-let`, `var-declaration`, `typeof-string`, `eq-null`, `no-await-in-loop`), TypeScript (3: `tuple-return`, `no-any`, `no-non-null-assertion`), Python (7: `print-debug`, `breakpoint`, `tuple-return`, `module-assign`, `bare-except`, `mutable-default-arg`, `assert-in-non-test`), Go (7: `fmt-print`, `many-returns`, `package-var`, `error-ignored`, `empty-return`, `defer-in-loop`, `context-todo`), Ruby (5: `binding-pry`, `global-var`, `rescue-exception`, `puts-in-lib`, `string-concat`), and cross-language (3: `no-todo-comment`, `no-fixme-comment`, `hardcoded-secret`).
