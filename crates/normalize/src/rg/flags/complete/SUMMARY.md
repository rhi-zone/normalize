# src/rg/flags/complete

Shell completion generators for the embedded ripgrep CLI, vendored from ripgrep 14.1.1. Generates completion scripts for Bash (`bash.rs`), Fish (`fish.rs`), Zsh (`zsh.rs`), and PowerShell (`powershell.rs`). An `encodings.sh` helper lists supported character encodings for completion. These are used by `normalize rg --generate complete-<shell>`.
