# src/service

Server-less `#[cli]` service layer — the primary CLI registration point. Each sub-module defines a service struct with methods annotated `#[cli(display_with = "...")]` that become subcommands. The `NormalizeService` root struct composes sub-services: `AnalyzeService`, `ConfigService`, `DaemonService`, `EditService`, `FactsService` (CLI name `structure`), `GrammarService`, `GuideService`, `GenerateService`, `PackageService`, `RankService`, `RatchetService`, `RulesService` (from `normalize-rules`), `ServeService`, `SessionsService`, `SyntaxService`, `ToolsService`, `ViewService`. Helper methods live in separate impl blocks; JSON/schema output is generated automatically by the proc macro.

Report type naming convention: all report structs use the `Report` suffix (e.g. `GuideReport`, `FileListReport`, `DaemonRunReport`, `TraceReport`). `DaemonRunReport` is the return type of `daemon run` (foreground execution); `DaemonActionReport` covers lifecycle actions (start/stop); `DaemonRootReport` covers root management (add/remove).

`RankService` (`rank.rs`) hosts all ranked-list commands (complexity, size, ceremony, hotspots, coupling, ownership, test-ratio, budget, density, uniqueness, imports, surface, depth-map, layering, module-health, call-complexity, duplicates, fragments, contributors). `ViewService` (`view.rs`) hosts `view` plus graph navigation subcommands (`callers`, `callees`, `history`, `dependents`, `trace`, `graph`, `provenance`); `trace()` returns `TraceReport`. `ConfigService` (`config.rs`) provides `normalize config schema/show/validate/set` with JSON Schema validation and `toml_edit` typed writes. `SessionsService` (`sessions.rs`) supports `--mode` (interactive/subagent/all) on list/stats/messages/analyze, plus the `subagents` subcommand. `RatchetService` (`ratchet.rs`) is a thin re-export of `normalize_ratchet::service::RatchetService`. The `normalize ci` method runs all three rule engines in sequence and exits non-zero on errors; flags `--no-syntax`, `--no-native`, `--no-fact`, `--strict`, `--sarif`.
