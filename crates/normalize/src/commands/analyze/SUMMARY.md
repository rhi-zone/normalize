# src/commands/analyze

All `normalize analyze` subcommand implementations (~43 files). Each file defines a report struct, implements `OutputFormatter` (`format_text` / `format_pretty`), and exposes a computation function. Representative subcommands: `complexity`, `length`, `hotspots`, `coupling`, `coupling_clusters`, `duplicates`, `docs`, `imports`, `layering`, `ownership`, `security`, `trace`, `trend`, `call_graph`, `graph`, `fragments`, `skeleton_diff`, `summary`. `mod.rs` declares all modules and defines the shared `AnalyzeConfig` struct. Native rule checks (`stale_docs`, `stale_summary`, `check_refs`, `check_examples`) and the shared `gitignore_walk()` walker have been extracted into the `normalize-native-rules` crate. The `rules_cmd` module has been extracted to the `normalize-rules` crate.
