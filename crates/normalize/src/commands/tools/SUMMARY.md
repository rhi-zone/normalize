# src/commands/tools

`normalize tools` subcommand implementations for running external ecosystem tools (linters, formatters, test runners) via the normalize-tools integration layer. Modules: `lint` (run ecosystem linters, collect diagnostics), `test` (run project test suites). These commands act as thin CLI wrappers over `normalize-tools`, surfacing external tool output as normalized `DiagnosticsReport` values. Both `lint run` and `test run` support `--repos-dir` / `--repos-depth` for multi-repo aggregation, running repos in parallel via rayon and collecting per-repo results into `RepoLintResult`/`RepoTestResult` vectors embedded in the top-level report.
