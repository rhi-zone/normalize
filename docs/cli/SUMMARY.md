# CLI Documentation

Reference documentation for each normalize CLI command, with one file per subcommand or command group. Covers the three core primitives (view, edit, analyze), search (grep), structural commands (structure, rules, plans), editor integrations (daemon, serve), and utility commands (init, grammars, update, generate, test). `README.md` provides the CLI architecture overview and command index. Kept in sync with the service layer and `args.rs` as commands are added or renamed.
