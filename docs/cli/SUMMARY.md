# CLI Documentation

Reference documentation for each normalize CLI command, with one file per subcommand or command group. Covers the core primitives (view, edit, analyze, rank, trend), search (grep), structural commands (structure, rules, plans), editor integrations (daemon, serve), and utility commands (init, grammars, update, generate, test, ci). `README.md` provides the CLI architecture overview and command index. `rank.md` documents the ranked-list commands migrated from `analyze`. `trend.md` documents the time-series health metrics migrated from `analyze` (2026-03-28). Kept in sync with the service layer as commands are added or renamed.
